    Как переустановить Windows с флешки Gigabyte

    В одной из предыдущих статей мы рассказывали о том как установить новую операционную систему на свой компьютер при помощи специальной утилиты Microsoft в режиме восстановления системы (Windows Recovery Environment). Однако это решение не всегда подходит для тех пользователей, которые хотят обновиться или же полностью изменить свою ОС, а так же удалить вирусы без потери данных и настройки программы. Для этого существует более продвинутое средство — USB-флешка под управлением гибридной BIOS/UEFI фирмы GIGABYTE, которая позволяет восстанавливать исходные заводские параметры компьютера, обновлять версию прошивки UEFI/BIOS, создавать дистрибутивы разных системных образов Windows и выполнять чистую установку выбранной вами версии ОС.

    Шаг первый: подготовка к работе

    • Скачайте последнюю стабильную сборку этой полезнейшей утилит со страницы производителя;
    • Вставьте ее содержимое на любую доступную вам USB-флешку емкостью от 2 ГБ минимум, но лучше использовать накопитель объемом больше чем 4 ГБ;
    • Перезагрузите ПК и войдите в режим Boot Menu нажатием клавиши F12 или Del во время запуска;
    • Выберите нужную вам запись с источником установки и нажмите Enter. Процесс должен зайтись автоматически. Если все прошло успешно то вы увидите экран приветствия с предложением выбрать тип работы инструмента. Нажмите “Start Now” для начала процедуры.

    Шаг второй: выбор режима обновления

    Откроется окно выбора типа выполнения действия. Здесь можно решить проблему зараженности вирусным кодом или просто очистить раздел жесткого диска от ненужных файлов. Выбираем вариант Refresh System и жмем Start Install чтобы продолжить процесс. После чего следует принять лицензионное соглашение и дать разрешение на использование стороннего ПО фирмой GigaByte (вы можете отказаться) после чего инструмент подготовит необходимые материалы для дальнейшего процесса установки новой версии OS. Далее нам предлагается сделать несколько действий по подготовке к процессу апгрейда. Мы рекомендуем внимательно ознакомиться с каждым пунктом и выполнить те рекомендации, которые показались разумными именно вашему пользователю. Например, если имеются важные данные хранящиеся на системе разделов HDD C:\ и D:\ то перед началом работы обязательно сделайте их резервиевую копию иначе вся информация будет удалена! А вот файлы расположенные на разделе E:\ могут оставаться там ведь они имеют статус временных и будут уничтожены вместе с ними весь установочный пакет который использовался для проведения данного мероприятия. Также рекомендуется проверить возможность использования данной технологии восстановление в случае возникновения проблем (несовместимости драйверов и т. д.) тогда стоит заранее узнать какие модели материнских плат поддерживают этот метод работы. И наконец убедитесь что ваш компьютер имеет достаточно памяти RAM (минимум 3Гб), свободного места на жестком диске (от 8Гб) и способен работать в безопасном графическом режиме (режим MS DOS может создать проблемы даже опытным специалистам!).

    Заключение

    Теперь остается только следовать подсказкам мастера инсталляции нового образа Windows и ждать завершения всех этапов преобразования вашего железухи и программного обеспечения. Обычно данный процесс занимает порядка часа времени и требует отключения любых внешних устройств хранения информации вроде Flash карт, подключенных носителей USB и сетевых принтеров поскольку любое вмешательство в работу инструмента приведет к его аварийному завершению. По окончанию всего комплекса операций происходит автоматическая перезагрузка системы где устанавливается новый образ вашей любимицы Windows XP / Vista / 7 / 8 / 8.1 либо 10 в зависимости от того какой комплект был предоставлен производителем компьютером или самостоятельно приобретенный пользователями набор необходимых пакетов программного обеспечения. Это отличная альтернатива стандартному методу модернизации своей рабочей станции путем ручного переноса содержимого раздела HDD С:\ в отдельный контейнер для последующей замены старой версии ОСи на свежую, особенно когда имеется дело с десктопом имеющим несколько физических накопителей. Ведь данная технология позволит вернуть машину в состояние «как было изначально» практически мгновенно без необходимости длительного ожидания окончания операции клонирования разделов посредством специализированных утилит.
