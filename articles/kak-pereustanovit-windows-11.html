Чтобы не терять важные файлы, заранее сделайте резервную копию всего на накопитель. А ещё можно воспользоваться одной из программ сторонних разработчиков для восстановления предыдущей версии операционной системы.

Процедура установки «Windows» может занять около часа или даже больше в зависимости от модели компьютера и объёма хранилища информации (SSD-накопители работают быстрее). В конце процесса система предложит пользователю настроить параметры конфиденциальности, сети, учётной записи Microsoft и внешнего вида рабочего стола. Это всё должно помочь вам быстро начать пользоваться компьютером после обновления, но если вы хотите удалить старую версию ОС, то перед этим обязательно создайте её копию — это позволит сохранить все данные пользователя. Также лучше скопировать всю информацию на внешний носитель.

Что делать перед обновлением Windows 11

  • Убедитесь, что компьютер соответствует системным требованиям новой OS. На веб‑сайте Microsoft есть специальный инструмент проверки совместимости. Он проверяет наличие необходимого процессора, памяти RAM и места хранения информации. Ссылка →
  • Скачайте последнюю сборку программы для создания загрузочной установочной флешки под названием Media Creation Tool. Её нужно скачать только один раз. После этого программа автоматически обновится во время работы. Для этого она использует интернет‑соединение вашего ПК. Загрузить утилиту со страницы поддержки Microsoft →
  • Создайте загрузочную USB‑флэшку. По умолчанию размер диска должен составлять минимум 8 ГБ. Если ваш накопитель меньше, то ему придётся разбиться на несколько разделов по мере того как будет идти процесс скачивания файлов. Недостаток свободного пространства приведёт к ошибке и прерыванию операции. Иначе говоря, ваша флешка должна соответствовать минимальным параметрам, указанным производителем. Вы можете проверить доступное пространство вручную, кликнув правой кнопкой мыши по диску C:\ и выбрав пункт “свойства”. Откроется окно с информацией о размере накопителя. Подробнее узнать про этот параметр и его влияние на скорость устанавливаемой операционной системы можно здесь. Сохраните содержимое жёсткого диска и SSD на другом устройстве. Перед началом процедуры удаления системы сохраните фото, видео, документы, музыкальные треки, игры и другие личные архивы. Кроме того, установите программу для синхронизации всех важных файлов между двумя ноутбуками. Так вы сможете получить доступ ко всем данным, когда будете использовать другой свой гаджет.

Как установить Windows 11 через флешку

Возьмите два флоппака – один нужен для сохранения текущего состояния вашей ОС, а второй – чтобы записать свежую установку операционки на нём. Первый шаг состоит именно в создании образа существующей системы. Когда он завершится, появится подсказка. Кликните кнопку Next. Затем следуя инструкциям мастера процедур, согласитесь с условиями использования ПО Microsoft и выберите папку для размещения готового файла. Важно! Оставьте настройки по умолчанию, иначе ничего работать не будет. Далее подтвердите начало процесса. Среднее время выполнения этой задачи составляет примерно час.

Теперь нужно создать буферный диск с помощью второго флеш‑носителя. Просто запустите ранее загруженную вами программу Media Creation Tool и нажмите Start Now. Установите флажок Use this PC to install Windows… (использовать этот компьютер для установки Windows…) и ждите окончания форматирования выбранного вами накопителя. Система предупредит вас обо всей предстоящей работе и покажет средний период времени, который потребуется на создание нового раздела. Скорее всего, понадобится более получаса. Закройте мастер и перезапустите систему. Через некоторое время откроются окна приветствия установщика Windows. Следуйте их подсказкам и выполняйте действия, которые будут предложены мастером.

Как вернуться к Windows 10?

Переход обратно к десятой версии возможен только в течение десяти дней после запуска новых функций. Однако важно помнить, что эта опция работает лишь тогда, когда новая версия операционной системы была активирована. Таким образом, если же ваши права доступа были приостановлены, то возможность возврата прежней системы исчезнет навсегда. Обратите внимание, что это касается именно перехода с «девятнадцатого» издания на однунадцатое. А вот отказ от новомодной осьмой версии станет проблемой, поскольку ей уже нет замены внутри самого продукта от Microsoft. Останется разве что установка какой‑нибудь альтернативной ОС вроде Linux, однако имеющиеся драйверы могут плохо взаимодействовать с ней. Лучше просто подождать выхода исправленной версии Windows 11.
