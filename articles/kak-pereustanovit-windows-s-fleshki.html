    В этом материале мы рассмотрим три способа восстановления системы на примере Lenovo IdeaPad B340-15IID. Но они подходят для любых других устройств тоже.

    Как начать восстановление компьютера из BIOS

    Это самый простой способ вернуть устройство к заводским настройкам или загрузить новую операционную систему при наличии установочного диска в формате ISO. Если у вас нет внешнего носителя информации, вы можете скачать образ ОС прямо во время работы текущей версии Windows. Сделать это можно через встроенный инструмент «Центр обновлений». Затем сохранить файл установки на другой диск (например, внешний жёсткий). И уже использовать его в качестве источника файлов для BIOS.

    • Перейдите в меню «Пуск» → Настройки → Обновление безопасности и центр управления → Восстановление -> Развёрнутый доступ к инструментам восстановления → Загрузка образа Windows → Выберите нужную версию ОС. Обратите внимание, что процесс скачивания занимает некоторое количество времени — около получаса. Также учтите размер файла — он может превышать несколько гигабайтов. Поэтому вам понадобится достаточно места на диске C:\Windows\SystemeRestore\.
    • После окончания загрузки нажмите кнопку “Сохранить” и укажите местоположение для сохранения образов OS. Откройте полученные файлы через архиватор WinRAR или аналогичную программу. Скопируйте содержимое одного из них на USB‑флэшку объёмом не менее 8 ГБ. Переименуйте её в другое имя (чтобы система поняла, что перед вами именно источник данных). Для этого щелкните правой кнопкой мыши по диску и выберите пункт Change Drive Letter and Volumes...→ Assign a drive letter to this volume…→ Next. Укажите букву D:.
    • Вставьте накопитель в один из портов вашего ПК и перезагрузите машину. После появления экрана заставка Lenovo нажмите F2 и войдите в раздел Boot Menu. Из списка выберите соответствующий элемент (“Boot from UEFI CD/DVD”). Здесь же следует настроить порядок запуска так, чтобы ваш флеш‑накопитель стоял первым. По желанию включите Fast Startup. Когда появится экран выбора языка интерфейса, нажмите Enter. Теперь просто следуйте инструкциям мастера установщика. Он предложит выбрать тип операции: установка новой копии Windows, восстановление предыдущих версий программного обеспечения, создание системных точек или полное удаление разделов. Рекомендуется первый вариант. Следующее окно содержит предложения установить некоторые программы сразу после завершения процесса. Обычно их пропускают, но если хотите ознакомиться с ними, сделайте необходимые настройки. Далее мастер попросит указать диск для размещения Windows. Можно создать новый том или оставить всё как есть. Окончание происходит автоматически. Перед тем как отключить питание машины, дождитесь полной инициализации всех служб.

    Что делать, если восстанавливать ОС нужно будет позже?

    Скачать необходимый образ можно прямо из раздела Recovery Tools внутри диспетчера задач. Однако этот метод работает только для пользователей корпоративных лицензий Microsoft. Чтобы получить стандартный ISO‑файл самой последней версии ОС, лучше воспользоваться официальным сайтом производителя оборудования вашей техники. Например, вот ссылка на страницу загрузки Image for ThinkPad and Yoga devices. Там найдётся всевозможная документация и инструменты, включая ссылки на страницы загрузки различных версий Windows, которые соответствуют модели вашего аппарата. Процесс получения доступа к файлу очень прост: заполняете форму, подтверждаете соглашение об использовании ПО, а затем выбираете путь для хранения файла установки. Вот ещё пара вариантов действий:

    * На странице Download Software for Your Device найдите вкладку File Browser. Кликнув по ней, откроется список актуальных версий ос, соответствующих вашему устройству. Доступны две опции загрузки: обычная версия со всеми программами или минимальная версия, включающая только самые основные функции. Первая займет больше пространства на диске, вторая экономичнее расходует ресурсы аппаратной части. Второй способ подразумевает использование специального портативного приложения Media Creation Tool для создания загрузочной флешки. Она позволит быстро устанавливать различные варианты Windows на любой подходящий ПК. А также содержать дополнительные компоненты для упрощения миграции пользовательских аккаунтов между разными машинами.

    Получив установочный образ, сохраните его на другом разделе жесткого диска либо подключенном внешнем носителе информации. Не забывайте сделать резервное копирование важных личных данных и фотографий заранее! Все инструкции для дальнейшей установки описаны выше. Или взгляните сюда, где подробно разобрано, как правильно выполнять такие действия.
