    Иногда программа не видит время на компьютере. Вот что делать в этом случае.

    Программа для общения Discord использует данные о времени вашего ПК, чтобы показывать вам онлайн-статус друзей или часы работы сервера — то есть сколько уже прошло с момента запуска канала. Впрочем, у некоторых пользователей возникают неполадки со временем чата и отображением статуса контактов. Это может раздражать, но легко исправляется. Следуйте инструкциям ниже, если столкнулись с подобной проблемой. 

    Если вы хотите узнать текущее время без необходимости открывать приложение

    Узнать актуальное положение стрелок можно прямо из меню уведомлений Windows (там же где находятся значки Wi‑Fi, громкости компьютера и прочие настройки). Нажмите на кнопку «Все действия» → «Настройки», затем переключитесь во вкладку «Приложения». Здесь найдите пункт “Discord” и нажмите на него. Выберите опцию синхронизации времени программы со временем системы. Теперь при запуске приложения статус будет обновляться автоматически.

    • Если нужно сделать это вручную, откройте раздел настроек Discord через шестерёнку справа снизу экрана клиента. Переходите по пунктам слева: Часы / Clock → Синхро­низировать / Synchronize time with your system. Повторите эти шаги снова после внесения изменений в системное время. Так гарантируется корректная работа функции часов, даже когда их параметры меняются.
    • Другой вариант — выбрать автоматическое определение местоположения, как мы описывали ранее. Для этого также зайдите в отдел настроек Discord, а потом в правую колонку с параметрами сервиса, там находится пункт Местоположение/Location. Откройте его. Найдите разделы Поддерживаемые страны / Supported Countries и Серверы близко к моей локации / Servers close to my location. Измените выбор ближайших серверов так, чтобы они были размещены рядом с вами географически. Тогда программа всегда покажет правильные значения времени, независимо от того, какой регион указан в вашем аккаунте. Активировать функцию определения региона можно в разделе Параметры пользователя / User Settings > Редактор профиля / Edit Profile -> Региональная служба доставки / Region and Timezone. Там необходимо поставить галочку напротив пункта Автоматический поиск местоположения / Automatically detect region from IP address.

    Что делать, если индикатор времени работает некорректно

    Возможно, ваш компьютер показывает неправильную дату или время? Например, отстаёт на несколько дней. Или, наоборот, идёт вперёд быстрее всех остальных устройств в сети. Доступны два способа решения этой ситуации. Первый подразумевает корректировку параметров дисплея в настройках операционной системы. Второй — перезагрузка устройства. Не спешите выбрасывать свой гаджет! И тот и другой метод работают безотказно. Просто следуйте нашим рекомендациям.

    Первым делом запустите программу установки Windows Update Utilities. Запустив её, дождитесь загрузки инструментария обновления ОС, расположенного в папке C:\Windows\SoftwareDistribution\Download. После завершения процесса кликните правой кнопкой мыши на файле WindowsUpdate_23H10-x86_en-US.msi и вызовите контекстное меню. Затем выполните команду Сохранение цели… / Save Target As…, находящуюся в нижней части списка. Кликните ещё раз правой кнопкой мышки по созданному файлу и добавьте его содержимое архиву ZIP. Сделаем это для удобства последующего использования.

    Скопируйте созданный архив в корневой каталог диска D:, либо E:. Делайте резервную копию важных данных перед этим действием. Распакуйте файлы из архива. Создание новой папки не требуется; всё должно работать сразу из распакованных файлов. Перед началом процедуры убедитесь, что все сторонние антивирусные продукты отключены. Также стоит проверить наличие свежих версий драйверов видеокарты и сетевого адаптера.

    Используя полученный инструмент, обновим файл SystemTimeOffset.reg из состава реестра OS. Дважды щелкните мышью по ярлыку WindowsUpdateUtilities.exe и активируйте главное окно. Щёлкните клавишами Ctrl + S, одновременно удерживая Shift. Появится диалоговое окно ввода командной строки. Важно ввести текст команды точно. Мы предлагаем следующий шаблон. Команда устанавливает системный таймер на значение UTC+07:00. Этот параметр
