    В браузере «Яндекс» есть функция режима инкогнито — это способ просматривать сайты без сохранения истории посещений и другой информации о вашем поведении. Вы можете быстро его активировать при помощи специальных комбинаций клавиш. 

    Как открыть режим инкогнито через горячие клавиши

    • Нажмите Ctrl + Shift (или Command + Shift) + N для Windows/Mac соответственно;
    • нажмите Alt+Shift+N для ChromeOS;
    • нажмите Cmd+T, а затем выберите вкладку “Новая” из появившегося меню.

    После этого откроется новая приватная страница. Если у вас уже открыты другие окна браузера, они останутся прежними и не будут затронутыми переключением режимов просмотра страниц. Они также смогут видеть друг друга. Но когда вы закончите использовать функцию защиты конфиденциальности, все ваши данные снова исчезнут. Поэтому будьте внимательны к тому, что делаете.

    Есть один дополнительный вариант активации функции, который имеет смысл использовать редко. Переместите курсоровую стрелку мышью либо пальцем так, чтобы она оказалась на одном из нижних углов экрана. Тогда там появляется контекстное меню выбора элементов интерфейса операционной системы. Активируйте команду Show Hidden Tabs (“Показать скрытые вкладки”) из этого меню. Все эти возможности очень полезны тем пользователям компьютеров, которые заботятся об охране своих персональных данных.
