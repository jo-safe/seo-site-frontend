    Как создать РГ на сервере в Minecraft

    Вы уже давно играете в Майнкрафтище? Тогда вам пора открыть собственный мир! Это будет ваш маленький райский уголок или же большая песочница для ваших друзей. Сделать это не так сложно, как кажется. Мы расскажем о создании сервера самостоятельно. Запускайте свой первый мир!

    Во многом похожие друг на друга версии игры отличаются только способом их получения (платная лицензия против бесплатного клиента) и тем самым отсутствием возможности устанавливать дополнительные моды и модифицировать игру по своему усмотрению. Но есть одна особенность — все они имеют возможность подключения к собственному миру через интернет. Если вы хотите собрать свою компанию игроков вокруг одного стола без необходимости искать подходящий онлайн-сервер, то вам придётся немного повозиться с установкой собственного игрового мира. Сейчас мы подробно разберём эту тему, чтобы каждый мог легко установить себе отдельный игровой мир. Для примера возьмем версию игры «Майнкрафт» под Windows, но если кто-то читает эти строки из Mac OS X, Linux или Android, ничего особого делать ему не нужно — просто следуйте этой инструкции.

    Подготовка

    • Установите Java Development Kit;
    • Создайте папку, куда будут загружаться файлы вашего будущего сервера. Назовите её server_folder;
    • Скачайте архивный файл с официального сайта «Майнкрафта»;
    • Распакуйте этот архив в созданную вами папку.

    Всё готово, можно приступить к делу. Переходим к первому шагу настройки нашего мировского королевства. Создаём его вместе!

    Первый шаг

    В распакованной папке найдите исполняемый файл minecraft\_server.jar. Дважды кликните по нему правой кнопкой мыши и выберите пункт «Открыть с помощью». Затем следует выбрать программу java (она наверняка будет находиться среди приложений). И вот он — чёрное окошко консоли с надписью «[INFO] [minecraft]: Starting mcservice192.168.0.10/1937…» Поздравляем, наш сервис начал работать. Теперь нам необходимо задать основные правила поведения в нашем мире.

    Второй шаг

    Откроем текстовый редактор (в Windows подойдёт Блокнот), пропишем туда следующее содержимое:

    enable‑command‑block=true gamemode=creative level‑type=flat world‑name=world allow‑flight=true spawn‑protection=4 max‑players=10 difficulty=easy announce‑player‑achievements=false whitelist=never broadcast‑user‑join‑messages=false disable‑mob‑spawning=true prevent‑end‑orid=<binary data> generate‑structures=false force‑gamemode=survival game‑rules.enableCheats=true

    Сохраните полученную строку в файле eula.txt. Прочитайте внимательно весь список правил, описанных выше, и решите сами, какие именно правила вы хотели бы оставить такими, а какие изменить. Обязательные пункты следующие:

    • «enable‑command‑block=true», необходимый параметр для использования командных блоков;
    • «level‑type=flat», определяет тип генерируемого ландшафта;
    • «spawn‑protection=4», количество клеток защиты точки появления игроков;
    • «allow‑flight=true», разрешение полётов;
    • «difficulty=easy», лёгкий уровень сложности;
    • «max‑players=10», максимальное число участников игры одновременно;
    • «whitelist=always», обязательно указывается имя каждого участника заранее;
    • game‑rules.enableCheats=true
    • Загологоловок: Третий шаг

    Закройте окно программы java. Папка server\_folder теперь содержит ещё один важный файл — run.bat. Кликните по нём два раза левой кнопкой мышки. Должно появиться новое окно с запросом на подтверждение запуска программного обеспечения. Не волнуйтесь, всё идёт правильно. Щёлкнув ОК, мы запустим нашу игровую среду. После этого откроется привычное интерфейсное меню, где нас ждёт новая панель управления нашим миром. Здесь мы можем менять многие свойства игры, добавляя новые элементы взаимодействия между нами и окружающим пространством. Также здесь присутствует ссылка на подробные характеристики каждой функции панели. Важный момент тут такой: после внесения изменений нажмите кнопку Save on Disk, иначе ваши изменения пропадут. Разумеется, в таком случае лучше сохранить оригинальные значения всех параметров.
